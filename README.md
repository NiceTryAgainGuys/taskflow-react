# TaskFlow

TaskFlow is a detailed React daily planner for organizing tasks, saving a planner file, importing task lists, and tracking work through a board-style workflow.

## Features

- Add tasks with a title, details, category, priority, and due date
- Save tasks in local storage so the list stays after refresh
- Export the full planner as a reusable JSON file
- Import a saved TaskFlow planner file
- Paste bulk tasks in a simple import format
- Export a readable text summary for sharing or saving notes
- Mark tasks complete or move them back to open
- Search across task titles, notes, and categories
- Filter the board by priority, category, and status
- Move tasks through Backlog, In Progress, Review, and Done lanes
- Track total, open, completed, high-priority, due-soon, and percentage-complete stats
- Highlight the next task that needs attention
- Clear completed work when the list gets crowded
- Responsive layout for desktop and mobile screens

## Tech Stack

- React
- Vite
- CSS
